# STA-terraform

A terraform-oriented image for StopThatAstronaut/The Devops Company

## Contains

- Terraform latest
- Azure CLI latest
- Powershell Core
- AWS CLI latest
- OpenSSH client (useful in the ror-aws-demo project)

## Build

CircleCI now set up. Builds images in every branch. Only pushes on main.
